# QassemFolio

Generate stunning developer portfolios directly from GitHub in seconds.

## Features

- ⚡ One-command portfolio generation
- 🎨 Beautiful modern templates
- 📊 GitHub stats integration
- 🚀 Easy deployment
- 🤖 AI-ready structure

## Quick Start
```bash
npx qassemfolio qassemadra
```
